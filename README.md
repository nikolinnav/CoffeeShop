# ☕️ Coffee Shop Website
🌐 Live Demo: https://nikolinnav.github.io/CoffeeShop/

## About the project
A cozy corner of the internet for coffee lovers - built to practice frontend fundamentals and make visitors want a latte. From glassmorphism header to the mobile-friendly navigation bar, every detail was crafted with care. 


## ✨Features
- Responsive navigation bar with hamburger menu animation  
- Glassmorphism-style header design  
- Background image optimization  
- Grid and flex-box based layout    
- Mobile-friendly design that feels good on every screen 


## 🛠️Built with
- HTML5  
- CSS3   
- JavaScript (Vanilla JS)


## 📱Responsive Design
The layout adapts gracefullyto any screen size using:
- Auto-adjusting grid layout  
- Mobile navigation menu  
- Flexible image scaling  
- Media query


## ⚡️Performance Optimization
A few thoughtful touches under the hood:
- Background image converted to WebP format  
- Lazy loading for non-critical images  
- Proper image compression and resizing  
- Layout stability using aspect-ratio property
